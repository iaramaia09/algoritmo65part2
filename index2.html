Algoritmo 65 - "Fatorial - para"
Fazer um programa que calcule o número fatorial de um número. Fatorial é a multiplicação
do próprio número por todos abaixo dele até 1.
Exemplo do fatorial de 5 seria: 5x4x3x2x1 = 120.
Fatorial de 5 é 120.

Algoritmo "Alg 65"
Var
num, num1, cont, cont1: inteiro

Inicio
escreva("Digite um número inteiro que deseja obter o fatorial: ")
leia(num)

num1 <- num
cont1 <- num

para cont de 2 ate cont1 faca
num1 <- num1-1
num <- num*num1
fimpara

escreva("O fatorial de ", cont1, " é ", num)
